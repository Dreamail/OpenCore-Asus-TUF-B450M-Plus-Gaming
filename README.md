# OpenCore-Asus-TUF-B450M-Plus-Gaming
Hackintosh EFI with OpenCore for Asus TUF B450M Plus Gaming

OpenCore Version: 0.6.3

Support MacOS 10.15.3-10.15.7 Catalina, 11.0.1 Big Sur

    - [x] Ryzen r5 2600
- [x] macOS 版本 / macOS Version

    - [x] 11.0.1 Big Sur
